PROGRAMME PPL — PUSH / PULL / LEGS
Guide complet pour pratiquants intermédiaires (6-12 mois d'expérience)
Auteur : FitCoach AI — Document de référence interne

=====================================================================
PRÉSENTATION DU PROGRAMME PPL
=====================================================================

Le programme Push/Pull/Legs (PPL) est une des organisations hebdomadaires
les plus efficaces pour les pratiquants intermédiaires. Il répartit les
groupes musculaires en 3 catégories selon leur fonction principale :

- PUSH (pousser) : pectoraux, épaules, triceps
- PULL (tirer) : dos, biceps, trapèzes
- LEGS (jambes) : quadriceps, ischio-jambiers, fessiers, mollets

Ce programme se décline sur 5 jours pour optimiser la récupération tout
en maintenant une fréquence d'entraînement de 2x par groupe musculaire
toutes les 2 semaines (version 5 jours) ou par semaine (version 6 jours).

=====================================================================
JOUR 1 — PUSH (Poussée — Lourd)
Muscles cibles : Pectoraux (chef sternal + claviculaire), Épaules antérieures, Triceps
=====================================================================

1. Développé couché barre (Bench Press)
   - Séries : 4 × 6-8 répétitions
   - Tempo : 3-1-1 (descente 3s / pause 1s / poussée 1s)
   - Repos : 3 minutes entre les séries
   - Note : exercice de base, prioriser la progression de charge (+1,25 kg toutes les 1-2 séances)

2. Développé incliné haltères (Incline Dumbbell Press)
   - Séries : 3 × 8-10 répétitions
   - Inclinaison : 30-45 degrés
   - Repos : 2 min
   - Note : cible le faisceau claviculaire des pectoraux

3. Dips pectoraux (Dips — version penchée)
   - Séries : 3 × 8-12 répétitions
   - Lest : ajouter du poids si plus de 12 reps deviennent trop faciles
   - Repos : 2 min
   - Variante : machine à contrepoids si non maîtrisé

4. Élévations latérales haltères (Lateral Raises)
   - Séries : 4 × 12-15 répétitions
   - Poids léger, contrôle du mouvement
   - Repos : 90 secondes
   - Note : ne pas tricher avec le buste — mouvement strict

5. Développé militaire haltères (Overhead Press)
   - Séries : 3 × 8-10 répétitions
   - Repos : 2 min
   - Note : alternative au développé militaire barre

6. Extension triceps poulie haute (Triceps Pushdown)
   - Séries : 3 × 10-12 répétitions
   - Poignée droite ou corde
   - Repos : 90 secondes

7. Kickback triceps (optionnel)
   - Séries : 2 × 12-15 répétitions
   - En superset avec extension poulie pour finir les triceps

=====================================================================
JOUR 2 — PULL (Tirage — Lourd)
Muscles cibles : Grand dorsal, Trapèzes, Biceps, Rhomboïdes, Ischio-jambiers (léger)
=====================================================================

1. Tractions pronation (Pull-ups)
   - Séries : 4 × max répétitions (ou 6-8 si avec lest)
   - Repos : 3 minutes
   - Note : exercice roi du dos. Utiliser une bande élastique si nécessaire.
   Objectif long terme : 4 × 10 avec poids de corps

2. Rowing barre (Barbell Row)
   - Séries : 4 × 6-8 répétitions
   - Buste à 45 degrés, tirage vers le nombril
   - Repos : 3 minutes
   - Note : exercice de base pour l'épaisseur du dos

3. Tirage horizontal câble (Seated Cable Row)
   - Séries : 3 × 10-12 répétitions
   - Poignée neutre (paumes face à face)
   - Repos : 2 minutes

4. Face pull (Face Pulls)
   - Séries : 4 × 15-20 répétitions
   - Poulie haute avec corde
   - Tirage vers le visage, coudes hauts
   - Repos : 90 secondes
   - Note : excellent pour la santé des épaules et le développement des faisceaux postérieurs

5. Curl biceps barre EZ (EZ Bar Curl)
   - Séries : 3 × 8-10 répétitions
   - Mouvement complet, pas de balancement
   - Repos : 2 minutes

6. Curl marteau haltères (Hammer Curl)
   - Séries : 3 × 10-12 répétitions
   - Prise neutre pour cibler le brachial
   - Repos : 90 secondes

7. Shrug haltères (Haussements d'épaules)
   - Séries : 3 × 12-15 répétitions
   - Pour développer les trapèzes supérieurs
   - Repos : 90 secondes

=====================================================================
JOUR 3 — LEGS (Jambes — Lourd)
Muscles cibles : Quadriceps, Ischio-jambiers, Fessiers, Mollets
=====================================================================

1. Squat barre (Back Squat)
   - Séries : 4 × 6-8 répétitions
   - Descente jusqu'au parallèle minimum
   - Repos : 3-4 minutes
   - Note : exercice fondamental. Progression : +2,5 kg toutes les 1-2 séances.
   Utiliser une ceinture de force au-delà de 1,5× poids de corps.

2. Leg Press (Presse à jambes)
   - Séries : 3 × 10-12 répétitions
   - Pieds espacés à largeur d'épaules, position haute pour les fessiers
   - Repos : 2 minutes

3. Fentes marchées haltères (Walking Lunges)
   - Séries : 3 × 12 répétitions par jambe
   - Repos : 2 minutes
   - Note : excellent pour l'équilibre musculaire et les fessiers

4. Leg Curl couché (Lying Leg Curl)
   - Séries : 4 × 10-12 répétitions
   - Ischio-jambiers en contraction complète
   - Repos : 2 minutes

5. Leg Extension (Extension quadriceps)
   - Séries : 3 × 12-15 répétitions
   - Repos : 90 secondes

6. Mollets debout (Standing Calf Raises)
   - Séries : 4 × 15-20 répétitions
   - Amplitude complète : descente maximale puis pointe des pieds
   - Repos : 60 secondes

7. Hip Thrust (Poussée de hanche)
   - Séries : 3 × 10-12 répétitions
   - Barre sur les hanches, dos contre un banc
   - Note : meilleur exercice pour l'activation des fessiers

=====================================================================
JOUR 4 — PUSH LIGHT (Poussée — Volume)
Muscles cibles : Pectoraux (faisceau interne), Épaules, Triceps
=====================================================================

1. Développé couché haltères (Dumbbell Bench Press)
   - Séries : 3 × 12-15 répétitions
   - Poids plus léger qu'en jour 1, focus sur la contraction

2. Écarté haltères incliné (Incline Dumbbell Fly)
   - Séries : 3 × 12-15 répétitions
   - Bon étirement en bas, contraction en haut

3. Crossover câbles (Cable Crossover)
   - Séries : 3 × 15 répétitions
   - Excellent pour l'isolation pectorale

4. Élévations frontales (Front Raises)
   - Séries : 3 × 12-15 répétitions
   - Alterner les bras ou simultané

5. Oiseau inversé haltères (Rear Delt Fly)
   - Séries : 3 × 15 répétitions
   - Buste penché, bras en croix

6. Triceps à la poulie basse (Overhead Triceps Extension)
   - Séries : 3 × 12-15 répétitions

=====================================================================
JOUR 5 — PULL LIGHT (Tirage — Volume)
Muscles cibles : Dos (épaisseur + largeur), Biceps, Arrière des épaules
=====================================================================

1. Tirage vertical poulie (Lat Pulldown)
   - Séries : 4 × 12-15 répétitions
   - Prise large, tirer vers le menton

2. Rowing haltère unilatéral (One-Arm Dumbbell Row)
   - Séries : 3 × 12 répétitions par côté
   - Appui sur un banc, amplitude maximale

3. Pull-over haltère (Dumbbell Pullover)
   - Séries : 3 × 12-15 répétitions
   - Allongé transversalement sur le banc

4. Curl incliné haltères (Incline Dumbbell Curl)
   - Séries : 3 × 12 répétitions
   - Étirement biceps en position inclinée

5. Curl concentration (Concentration Curl)
   - Séries : 3 × 12-15 répétitions par bras

6. Face pull léger + élévations latérales (Superset)
   - 3 × 15 répétitions chacun
   - Récupération minimale (30 secondes) entre les exercices

=====================================================================
REPOS ET RÉCUPÉRATION
=====================================================================

Jours de repos recommandés :
- Programme 5 jours : Lun-Mar-Mer-Jeu-Ven + Week-end repos
- Alternative : Jour1-Jour2-Repos-Jour3-Jour4-Jour5-Repos

Durée recommandée des séances : 60 à 75 minutes maximum.
Au-delà, le niveau de cortisol augmente et nuit à la récupération.

Sommeil : 7-9 heures par nuit — indispensable pour la synthèse protéique.
Hydratation : minimum 35 ml par kg de poids de corps par jour.
Exemple : 80 kg → 2,8 litres d'eau/jour minimum.

=====================================================================
NUTRITION — CONSEILS DE BASE
=====================================================================

APPORT EN PROTÉINES
- Objectif minimal : 1,6 g de protéines par kg de poids de corps par jour
- Objectif optimal : 2,0 g/kg (prise de masse) à 2,2 g/kg (sèche)
- Exemple pour 80 kg : 160 g à 176 g de protéines par jour
- Sources : poulet, thon, œufs, fromage blanc 0%, whey protéine

RÉPARTITION CALORIQUE RECOMMANDÉE
- Protéines : 30-35% des calories totales
- Glucides : 40-50% des calories totales
- Lipides : 20-25% des calories totales

TIMING NUTRITIONNEL
- Repas pré-entraînement (2h avant) : glucides complexes + protéines
  Exemple : 150g de riz + 150g de blanc de poulet
- Repas post-entraînement (dans l'heure) : protéines + glucides simples
  Exemple : Whey + banane ou riz blanc + filet de poisson
- Ne pas s'entraîner à jeun si l'objectif est la performance

SUPPLÉMENTATION DE BASE (optionnelle)
- Créatine monohydrate : 3-5g par jour, à n'importe quel moment
  Effet : +3-8% de force, amélioration de la récupération musculaire
  Études : plus de 500 études cliniques confirment son efficacité et sa sécurité
- Whey protéine : complément pratique si les apports alimentaires sont insuffisants
- Vitamine D3 : 1000-2000 UI par jour, surtout en hiver
- Oméga-3 : 2-3g d'EPA/DHA par jour pour réduire l'inflammation

GESTION CALORIQUE SELON L'OBJECTIF
- Prise de masse : surplus de 200-400 kcal/jour au-dessus du TDEE
  → Viser 0,5-1 kg de prise de poids par mois (au-delà = prise de gras excessive)
- Sèche : déficit de 300-500 kcal/jour en dessous du TDEE
  → Viser 0,5-1% de poids de corps perdu par semaine
- Maintien : calories = TDEE, protéines élevées pour préserver la masse musculaire

=====================================================================
SUIVI ET PROGRESSION
=====================================================================

MÉTHODES DE PROGRESSION
1. Surcharge progressive : augmenter la charge dès que vous réalisez
   le haut de la fourchette de répétitions sur toutes les séries
   Exemple : si vous faites 4 × 10 au lieu de 4 × 6-8 → augmenter la charge

2. Double progression : viser d'abord le bas de la fourchette en nombre de reps,
   puis augmenter les reps avant d'augmenter la charge

3. RPE (Rate of Perceived Exertion) : travailler entre 7 et 9/10
   → 8/10 = il vous reste 2 répétitions dans le réservoir en fin de série

INDICATEURS DE PROGRESSION À SUIVRE
- Poids de corps (peser le matin à jeun, moyenne sur 7 jours)
- Charges utilisées pour les exercices principaux (journal d'entraînement)
- Tour de bras, de poitrine, de taille (mensuel)
- Photos de progression (toutes les 4-6 semaines)
- Niveau d'énergie et qualité du sommeil

ERREURS FRÉQUENTES À ÉVITER
1. Négliger les exercices composés (squat, soulevé de terre, développé couché)
   au profit des exercices d'isolation
2. Manger insuffisamment en prise de masse (impossible de progresser sans surplus)
3. Changer de programme trop souvent (minimum 8-12 semaines par programme)
4. Négliger le sommeil et la récupération
5. S'entraîner avec des charges trop lourdes au détriment de la technique
